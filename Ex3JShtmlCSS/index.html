Brief description of Graph: Bar Plot

A boxplot is a method for displaying data with summary values, such as first quartile (Q1), median, third quartile (Q3), max and min values. It is useful to identify outlier values in the data that lie beyond the quartiles.

Brief description of Graph: Violin Plot

A violin plot displays the summary values of the box plot, such as first quartile (Q1), median, third quartile (Q3), max and min values. Additionally violin plot displays the distribution of data points.

Brief description of Graph: Violin Plot with Bar plot and raw data Scatter Plot

A violin plot displays the summary values of the box plot, such as first quartile (Q1), median, third quartile (Q3), max and min values. Additionally violin plot displays the distribution of data points. In this graph box plot is shown inserted inside the violin plot and scatter of raw data points are shown to illustrate this.
